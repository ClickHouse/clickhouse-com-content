Columns:

  • role_name (Nullable(String)) ‚Äî Role assigned to user account.

  • access_type (Enum8) ‚Äî Access parameters for ClickHouse user account.

  • database (Nullable(String)) ‚Äî Name of a database.

  • table (Nullable(String)) ‚Äî Name of a table.

  • column (Nullable(String)) ‚Äî Name of a column to which access is granted.
